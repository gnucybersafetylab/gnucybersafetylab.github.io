RoboRacer Build 개요

이 섹션은 실제 RoboRacer 차량을 조립하고, NVIDIA Jetson 및 주변 장치를 설정하며, 드라이버 스택을 설치하고 테스트하는 전체 과정을 다룹니다.

목표는 다음과 같습니다.

  • 섀시, 전자 부품, 자율주행 센서를 포함한 전체 차량 하드웨어를 완성한다.
  • Jetson 보드와 LiDAR, 카메라, VESC 등의 주변 장치를 안정적으로 연결한다.
  • RoboRacer 전용 드라이버 스택과 네트워크 설정을 완료한다.
  • 실제 트랙에서 주행 가능한 상태의 1/10 스케일 자율주행차를 구축한다.

1) Building the RoboRacer Car

이 단계에서는 Bill of Materials(BOM)를 기반으로 하드웨어를 조립합니다. 하부 섀시, 서스펜션, 조향, 구동계, 상부 자율주행 모듈(센서·컴퓨팅 보드)을 순서대로 구성하게 됩니다.

2) Configure Jetson and Peripherals

Jetson 보드에 OS를 설치하고, 네트워크 설정, SSH 접속, 개발 환경 세팅, 센서 드라이버를 구성합니다. 특히 VESC, LiDAR, 카메라와의 통신 설정이 중요합니다.

3) Install RoboRacer Driver Stack

ROS 기반의 RoboRacer 드라이버 스택과 관련 패키지를 설치하고, 시리얼/이더넷 포트, 토픽 이름 등을 실제 하드웨어에 맞게 튜닝합니다. Docker 환경을 사용하는 경우, 컨테이너 기반 설정도 함께 제공합니다.

4) Driving the RoboRacer Car

마지막으로, 기본 키보드/조종기 제어부터 자율주행 데모까지 실행해 보며 전체 빌드 과정이 정상적으로 완료되었는지 확인합니다.

2. RoboRacer - Course Documentation

빌드가 완료되면, 이제 RoboRacer 차량을 활용하여 자율주행 알고리즘을 학습할 차례입니다. 이 섹션에서는 강의 모듈, 실습(Labs), 레이스 과제, 최종 프로젝트 등 교육 과정 전반에 대한 안내를 제공합니다.

⚠️ 안내: 처음 시작하는 경우, 먼저 Getting Started 문서를 참고한 뒤 각 모듈을 순서대로 학습하는 것을 추천합니다.

교육 모듈 (Modules)

  • Module A: Introduction to ROS & RoboRacer
  • Module B: Reactive Methods (벽추종, 갭추종 등)
  • Module C: Mapping & Localization (SLAM, Localization)
  • Module D: Planning & Control (Pure Pursuit, MPC 등)
  • Module E: Vision (카메라 기반 인식)
  • Module F: Special Topics (강화학습, GPU 활용 등)
  • Module G: RoboRacer Grand Prix!! (종합 프로젝트 & 레이스)

실습 & 과제 (Assignments)

  • Labs – 각 모듈별 실습 코드와 노트북
  • Races – 트랙 주행 과제 및 랩타임 평가
  • Final Project – 팀 기반 종합 자율주행 프로젝트
  • Grading Rubrics – 평가 기준 및 제출 형식 안내
  • Material Downloads – 슬라이드, 코드, 데이터셋 다운로드
Note: RoboRacer 교육 자료는 오픈 소스로 운영되며, GitHub 이슈를 통해 버그 제보, 개선 제안, 새로운 아이디어를 언제든 공유할 수 있습니다.
